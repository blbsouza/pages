Instruções de instalação

Pré-requisitos para instalação e uso da aplicação.

  • Baixe a aplicação na Página de Apresentação
  • Descompacte a pasta "Brdoc" no diretório portal do srvsaveFFF.
  • Abra um navegador e chame a url https://srvsaveFFF/doclider/
  • Adicionar à rotina a execução diária da shell que vem junto com o programa

Problemas de permissão na pasta 'relatorios'

Quando os arquivos não são copiados para a pasta relatórios pode ser problemas de permissão. Para correção tente seguir os passos:

  • Através do terminal, acesse a pasta doclider que você copiou para o portal que está em srvsaveFFF/files/portal.
  • Entre na pasta cd doclider
  • Delete a pasta relatorios com rm -R relatorios
  • Crie uma nova pasta com mkdir relatorios
  • Dê permissão à pasta com chmod 777 relatorios

Tente executar a shell novamente ou copiar algum arquivo manualmente para a pasta.
