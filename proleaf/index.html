PROJETO PROLEAF

Este projeto é um conjunto de páginas que visam facilitar o dia-a-dia, entregar desempenho e que focam no respeito ao meio ambiente.

Veja a seguir o que preparamos para você

Clientes

Página que auxilia o televendas com a geração de tabelas com descrição do item, preço, quantidade e o código de barras.

Batimento

Página que auxilia na impressão de páginas para encadernamento onde é feito o registro de quebras/sobras dos operadores.
